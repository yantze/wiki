# Linux

linux 文件属性
```
# ls -al
# drwxr-xr-x   6 yantze  staff    204 Mar 15 16:23 kvm
d 文件夹
p pip 文件
- 普通文件
c char 设备，字节设备

-r--r--r-T.  T代表写保护文件 chmod +t

```

linux 更新方式
```
滚动式更新
Arch Linux，每次运行pacman -Syu 更新软件时，新的功能都会带到你的系统，不需等待新的发行版。 Slackware，Debian，Gentoo/Funtoo 是比较典型的滚动式更新

发行版更新
Ubuntu，系统的新特性只会在每个发行版发行时出现，同时发行版发行前几周就会冻结特性。
```
